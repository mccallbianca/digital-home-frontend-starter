# Banned Phrases

Words and phrases the AI must never use. These make content sound generic, corporate, or like every other ChatGPT output.

## Always Banned
- "In today's fast-paced world"
- "Game-changer"
- "Leverage" (as a verb)
- "Synergy"
- "Circle back"
- "Low-hanging fruit"
- "Move the needle"
- "At the end of the day"
- "It goes without saying"
- "Without further ado"
- "In conclusion"
- "Dive deep" / "deep dive"
- "Unlock your potential"
- "Take it to the next level"
- "Revolutionize"
- "Cutting-edge"
- "Best-in-class"
- "World-class"
- "Seamless" / "seamlessly"
- "Robust"
- "Holistic"

## Conditional (OK in specific contexts)
- "Ecosystem" — OK when used in a technical context, banned when generic filler
- "Journey" — OK when literal, banned as a metaphor for "process"

## Use Instead
| Banned | Replacement |
|--------|-------------|
| Leverage | Use |
| Utilize | Use |
| Optimize | Improve / fix |
| Paradigm shift | Change |
| Scalable | Grows with you |
